Anton Danylenko's Sudoku Solvers

Naive "Look for the next open cell (linearly), find and place the next non-conflicting guess, then go on, backtrack when there are no valid guesses for a cell" - Mr. Brooks
Smarter-1 Test each number individually. For every three-by-three square, check if the number can be placed in any of the open spaces. If it can only be placed in one space, the number is permanently put there. If not, nothing happens. The solver continues to the next square. Once all the squares have been tried, the solver applies the same strategy to the rows and columns. This is done for all the numbers. The solver repeats this process until all the definite numbers have been placed. It then resorts to the naive method.
